Observation Charts

Observation Charts is a JavaScript package to draw SVG star charts.

Observation Charts is a work in progress and subject to change without notice, rhyme, or reason.

Observation Charts can be used to draw full-sky charts for a given center RA/declination or for a given latitude/longitude at a given date and time:

<div id="full-sky"></div>
...
new ObservationChart(d3.select('#full-sky')[0][0], {
    size: {
        width: 800,
        height: 800,
    }
});

Observation Charts can also be draw only specific regions with a given center RA/dec and a particular scale/size configuration:

<div id="orion"></div>
...
new ObservationChart(d3.select('#orion')[0][0], {
    size: {
        width: 400,
        height: 400,
    },
    center: {
        ra: 5.8,
        dec: 0.0 
    },
    scale: 4,
    zoom: {
        zoomable: false
    }
});
